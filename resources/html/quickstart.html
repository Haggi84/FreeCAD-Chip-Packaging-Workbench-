Quick Start Guide

1. Load GDSII Files

  1. Click "Load GDSII" in the toolbar
  2. Select your .gds file
  3. Select matching .lyp file for colors
  4. Choose layers to import in the selection dialog
  5. Configure display options:
    • Match KLayout - Use LYP colors
    • Highlight Bondable - Gold coloring
    • Import All Layers - Quick selection

2. Create Leadframe

  1. Click "Leadframe Configurator"
  2. Choose package type:
    • QFN - Quad Flat No-lead
    • QFP - Quad Flat Package
    • BGA - Ball Grid Array
  3. Set dimensions and lead parameters
  4. Choose material
  5. Generate leadframe geometry

3. Stack Layers on Leadframe

  1. Click "Layer on Leadframe"
  2. Follow GDS import steps (select files and layers)
  3. Configure leadframe parameters
  4. Set transformation options:
    • Auto-fit - Scale to fit opening
    • Rotation - Arbitrary rotation
    • Mirror - Flip top/bottom
    • Offset - Precise positioning
  5. Generate stacked assembly

4. Create Housing

  1. Click "Housing Configurator"
  2. First configure leadframe using the button
  3. Set housing parameters:
    • Wall thickness and clearance
    • Housing height
    • Lid options and transparency
    • Material selection
  4. Generate transparent package

5. Manual Wire Bonding

  1. Click "Manual 2D Wire Bonding"
  2. Configure wire parameters in dialog
  3. Click on die pads (green markers appear)
  4. Click on bond fingers (blue markers appear)
  5. Repeat for additional bonds
  6. Use "Finish Wire Bonding" command to complete
