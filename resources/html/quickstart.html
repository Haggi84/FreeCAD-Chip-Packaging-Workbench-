🚀 Quick Start Guide

Follow these steps to go from a raw GDS file to a fully bonded chip-package assembly.

Step 1 — Load GDS Layout

  1. Click Load GDSII in the main toolbar
  2. Select your .gds file
  3. Select the matching .lyp KLayout color file
  4. Optionally select an IHP .map file for layer name resolution
  5. Optionally select a stackup .xml for accurate 3-D Z-heights
  6. In the layer selector choose which layers to show and set import options:
    • Match KLayout colors — use LYP fill/frame colors exactly
    • Highlight bondable — gold highlight for PIN/bond-pad layers
    • Extrude to 3-D — convert polygons to solids using stackup data
    • Auto PIN contacts — automatically place contact points on detected bond pads
Tip: Use Auto PIN contacts to skip manual contact-point placement for standard IHP PDK layouts — the tool reads DT=2 pin markers to find the exact bond locations.

Step 2 — Position the Chip

  1. Click Move / Rotate Chip in the toolbar
  2. Select scope: GDS Chip Objects moves only GDS layers, contact points and wires
  3. Use the arrow buttons (or keyboard ←→↑↓ / PgUp/PgDn) to translate
  4. Use the Rotate buttons (or Shift + arrow keys) to rotate around the bounding-box center
  5. Click Restore Original to undo all changes in the current session

Step 3 — Define Contact Points

Option A — Automatic (recommended)

  1. Enable Auto PIN contacts during GDS import (Step 1)
  2. Contact point markers (orange dots) appear on the detected bond pads

Option B — Face grid tool

  1. Click Set Contact Points on Face (main toolbar)
  2. Ctrl+click one or more pad faces to select them
  3. Click Generate Grid — a UV grid of candidate points appears on each face
  4. Click the desired grid points to select them (they turn orange)
  5. Click OK — selected points become contact point markers

Option C — Define Contact Points (advanced toolbar)

  1. Select one or more GDS layer objects in the 3-D view
  2. Click Define Contact Points — a marker is placed at the top-face centre of each selected object

Step 4 — Leadframe (Advanced Toolbar)

First click the Advanced Tools ⇄ button to reveal the advanced toolbar.

  1. Click Leadframe Configurator
  2. Choose package type: QFN, QFP, or BGA
  3. Set frame dimensions, lead count, pitch, and material
  4. Click Generate
  5. Use Center Leadframe to align the chip over the die-attach paddle

Alternatively load a real package STEP model via Leadframe Library.

Step 5 — Wire Bonding

  1. Click Wire Bond in the main toolbar (requires both GDS contact points and leadframe contact points)
  2. Configure wire diameter, arc height, and material in the dialog
  3. Click an orange die-side contact point, then a yellow housing/leadframe contact point
  4. A solid 3-D wire is created between the two points
  5. Repeat for each connection
  6. Click Cancel Wire Bonding when finished (or close the session)

Step 6 — Wire Bumps

  1. Click Wire Bump Configurator
  2. Select a bump shape: Ball Bond, Wedge Bond, Stitch Bond, or Nail Head
  3. Adjust geometry parameters — a live cross-section preview updates in real time
  4. In the Netlist table, check the connections you want to bump
  5. Click Place Bumps on Selected Connections — bumps appear at both wire endpoints

Step 7 — Save Session

  1. Click the Session ▾ button and choose Save Design Session
  2. Choose a filename — a .dipas JSON file is written alongside your FreeCAD document
  3. To replay, click Session ▾ → Load Design Session and select the .dipas file
