Troubleshooting Guide

Common Issues and Solutions

No Layers Found During Import

Possible Causes:

  • GDS and LYP files don't match
  • LYP file has all layers hidden
  • Files are corrupted or incomplete
  • Wrong file format selected

Solutions:

  • Verify GDS and LYP files are from same project
  • Check LYP file in KLayout - ensure layers are visible
  • Try importing without LYP file first
  • Use "Import All Layers" to see if any layers appear

GDS Import Fails

Possible Causes:

  • File path contains special characters
  • Insufficient memory for large files
  • GDS file contains no polygons
  • File permissions issue

Solutions:

  • Move files to simple path (no spaces/special chars)
  • Close other applications to free memory
  • Verify GDS file in KLayout first
  • Check file permissions and try different directory

Display/Visual Issues

Possible Causes:

  • Graphics driver compatibility
  • Transparency settings conflict
  • Color mode mismatch
  • Large file performance issues

Solutions:

  • Update graphics drivers
  • Try different color modes (KLayout vs Material)
  • Disable transparency for complex layouts
  • Use 2D preview mode for large designs

Wire Bonding Not Working

Possible Causes:

  • No valid die pad objects selected
  • No valid bond finger objects available
  • Selection mode confusion
  • Session not properly started

Solutions:

  • Ensure objects contain keywords: 'die', 'pad', 'metal' for die pads
  • Ensure objects contain keywords: 'lead', 'finger', 'terminal' for bond fingers
  • Check console messages for selection feedback
  • Restart wire bonding session if needed

Performance Tips

  • Use "Import All Layers" for quick previews, then refine selection
  • Enable decimation for very complex layouts with many polygons
  • Use 2D preview mode when working with large GDS files
  • Close property panel when not actively modifying layers
  • Import only needed layers rather than entire GDS file
  • Use material styling instead of KLayout matching for better performance

Debugging Steps

  1. Check Python Console for error messages and warnings
  2. Verify file compatibility by opening in KLayout first
  3. Test with simple files to isolate the issue
  4. Check FreeCAD version compatibility
  5. Look for workbench updates that might fix known issues

Getting Help

If issues persist:

  • Check the Python Console for detailed error messages
  • Verify your GDS and LYP files work in KLayout
  • Ensure you have the latest version of the workbench

  • Contact the development team with:
    • FreeCAD version information
    • Error messages from Python Console
    • Sample files that reproduce the issue
    • Steps to reproduce the problem
