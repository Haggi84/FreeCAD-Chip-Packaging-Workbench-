Tool Reference

GDSCommand - Load GDSII

Purpose: Import GDSII files with layer selection and styling

Inputs: GDS file, LYP file, optional MAP file

Outputs: 2D/3D geometry with material colors

Features:

  • Interactive layer selection
  • KLayout color matching
  • Material-based styling
  • Property panel integration

LeadframeCommand - Leadframe Configurator

Purpose: Create semiconductor package leadframes

Supported Types: QFN, QFP, BGA

Parameters:

  • Frame dimensions (length, width, thickness)
  • Lead counts (all sides)
  • Lead geometry (width, pitch, length)
  • Material selection

HousingCommand - Housing Configurator

Purpose: Generate transparent IC packages

Features:

  • Customizable wall thickness
  • Transparent lid options
  • Alignment posts
  • Material and transparency controls

Dependencies: Requires leadframe configuration

LayeronLeadframe - Layer on Leadframe

Purpose: Stack semiconductor layers on leadframes

Features:

  • Auto-fitting with margin control
  • Rotation and mirroring
  • Z-stacking based on technology
  • Combined GDS + leadframe workflow

WirebondCommand - Manual 2D Wire Bonding

Purpose: Create manual wire bonds between die and package

Workflow: Die pad selection → Bond finger selection

Features:

  • Object-based selection (no random clicks)
  • Straight and arc wire styles
  • Visual feedback with colored markers
  • Automatic object identification
  • Finish/Cancel session commands

Property Panel

Location: Right dock widget

Tabs:

  • Layer Properties - Raw LYP properties
  • Color Summary - Color usage statistics
  • Technology - IHP mapping information

Features:

  • Real-time layer information
  • Modification without re-import
  • Color and visibility controls
  • Technology layer mapping

FinishWireBondingCommand

Purpose: Complete wire bonding session

Features:

  • Generates detailed bonding report
  • Removes selection observer
  • Shows bond count summary
  • Cleans up temporary data

CancelWireBondingCommand

Purpose: Cancel current wire bonding session

Features:

  • Stops selection listening
  • Clears temporary data
  • Returns to normal mode
