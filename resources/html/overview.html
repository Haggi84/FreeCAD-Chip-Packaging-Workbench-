Workbench Overview

The DI-PASSIONATE FreeCAD Workbench provides comprehensive tools for semiconductor packaging design and IC layout visualization.

► Key Features

  • GDSII Import - Load and visualize semiconductor layout data
  • Leadframe Design - Create QFN, QFP, and BGA packages
  • Housing Design - Generate transparent IC packages
  • Layer Stacking - Stack semiconductor layers on leadframes
  • Wire Bonding - Manual 2D wire bonding capabilities

■ Supported File Formats

FormatExtensionPurpose
GDSII.gds, .GDSLayout geometry data
LYP.lyp, .LYPKLayout layer properties (colors)
MAP.map, .MAPIHP technology mapping (optional)

★ Visual Features

  • KLayout LYP file compatibility
  • Material-based styling
  • Real-time property panels
  • Layer modification capabilities
  • Bondable layer highlighting
