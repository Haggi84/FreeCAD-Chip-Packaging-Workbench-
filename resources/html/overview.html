🏠 Workbench Overview

The DI-PASSIONATE FreeCAD Workbench provides an end-to-end design environment for semiconductor chip-packaging — from raw GDS layout data all the way to a fully bonded 3-D assembly ready for export.

Design Flow at a Glance
Import GDS → Position Chip → Define Contact Points → Create Leadframe → Wire Bond → Configure Bumps → Export / Save Session

Key Features

FeatureDescription
GDSII ImportLoad semiconductor layout (.gds) with KLayout color files (.lyp) and optional IHP technology mapping (.map)
3-D ExtrusionConvert flat layout polygons to proper 3-D solids using PDK stackup data from a KLayout XML file
Chip TransformTranslate and rotate the entire chip assembly with arrow buttons or keyboard shortcuts
Contact PointsAuto-detect bond pads from PIN layers, or place them manually on faces with a grid-based tool
Leadframe DesignParametric QFN / QFP / BGA leadframes, or import STEP models from the IHP mirror library
Wire BondingManual 3-D wire bonding between die pads and package leads — produces solid swept-pipe geometry
Wire Bump ConfiguratorPlace Ball, Wedge, Stitch, or Nail-Head bumps at both ends of every bond wire
Session Save / LoadRecord the full design history and replay it from a .dipas JSON file

Supported File Formats

FormatExtensionRole
GDSII.gdsIC layout geometry
KLayout Layer Properties.lypLayer colors and visibility
IHP Technology Map.mapLayer name → EDI type mapping (optional)
KLayout Stackup XML.xmlAccurate Z-heights from PDK (optional)
STEP.step / .stpPackage models from mirror-semi library
DI-PASSIONATE Session.dipasReproducible design history (JSON)

System Requirements

  • FreeCAD 1.0 or newer
  • Python packages: gdstk (GDS reading), PySide2 (included with FreeCAD)
  • Optional: KLayout for generating .lyp and .map files
