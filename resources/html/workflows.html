Workflow B — GDS Visualization Only

  1. Load GDSII with Match KLayout colors enabled
  2. Select only the layers you need to inspect
  3. Enable Extrude to 3-D and provide the stackup XML for accurate heights
  4. Use FreeCAD standard view tools (isometric, section cuts) for inspection

Workflow C — Quick Bond Point Check

  1. Load GDSII with Contacts-only 3-D and Auto PIN contacts enabled
  2. Only bond-pad layers are shown as full geometry; the rest collapses to a single grey body box
  3. Inspect the auto-placed orange contact points to verify pad detection
  4. If any are wrong, delete them and use Define Contact Points to place manually

Workflow D — Session Replay

  1. Open FreeCAD with the DI-PASSIONATE workbench
  2. Click Session ▾ → Load Design Session
  3. Select the .dipas file
  4. All recorded actions are replayed automatically — GDS import, leadframe, housing, wire bonding

Session replay is useful for regenerating a design after modifying the source GDS file.

Workflow E — STEP Package Import

  1. Load GDS layout first
  2. Reveal the Advanced toolbar (click Advanced Tools ⇄)
  3. Click Leadframe Library
  4. Browse to the package STEP file
  5. In the orientation dialog, click the face that should face up
  6. In the die-attach dialog, click the die-attach paddle face
  7. The STEP model is placed automatically with the die centered on the paddle

Coordinate System

Units: Millimeters (FreeCAD standard)
GDS scale: 1 GDS user unit = 0.001 mm (standard µm convention)
Z orientation: +Z points up; chip die surface faces +Z; bond wires arc above the die

Typical chip size: 1–5 mm XY; 0.003–0.010 mm total stack height
Typical bond wire length: 0.5–3 mm

Tips & Best Practices

  • Always load the .map file if available — it enables accurate layer identification for Auto PIN detection and bondable highlighting
  • Use Contacts-only 3-D mode for large chips (>1 M polygons) to keep the viewport fast
  • The Contact Point Browser dock lets you quickly verify which pads have been marked
  • Each wire bond is a separate FreeCAD transaction — Ctrl+Z undoes individual wires
  • Move/Rotate Chip captures a placement snapshot on open; use Restore Original to undo all moves at once
  • Save the session before closing — the .dipas file is much smaller than a .FCStd file and fully reproducible
