Workflow Examples

Complete Semiconductor Package Design

  1. Import GDSII Layout
    • Use "Load GDSII" command
    • Select die layout file
    • Choose bondable layers (metals)
    • Apply material styling
  2. Create Leadframe
    • Use "Leadframe Configurator"
    • Select QFN/QFP package type
    • Set dimensions to match die size
    • Configure lead count and pitch
  3. Stack Die on Leadframe
    • Use "Layer on Leadframe" command
    • Enable auto-fit with margin
    • Set rotation if needed
    • Generate stacked assembly
  4. Add Wire Bonds
    • Use "Manual 2D Wire Bonding"
    • Click die pads → bond fingers
    • Use arc wires for longer spans
    • Finish session to generate report
  5. Create Housing
    • Use "Housing Configurator"
    • Set wall thickness and clearance
    • Configure transparent lid
    • Generate final package

Modification Workflow

  1. Update Layer Selection
    • Open Property Panel (right dock)
    • Click "Modify" button
    • Select different layers
    • Apply changes without re-import
  2. Change Leadframe Parameters
    • Use Property Panel modification
    • Update dimensions or lead count
    • Regenerate with new parameters
  3. Adjust Wire Bonds
    • Delete existing bond wires
    • Restart wire bonding session
    • Create new bond connections

Styling Workflow

  1. KLayout Color Matching
    • Import with "Match KLayout" enabled
    • Uses exact LYP file colors
    • Best for KLayout compatibility
  2. Material-Based Styling
    • Import with material styling
    • Colors based on layer type
    • Metals = gold, oxides = blue, etc.
  3. Bondable Highlighting
    • Enable "Highlight Bondable"
    • Bondable layers shown in gold
    • Quick identification for wire bonding
